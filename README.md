# Cloud United

### OpenClaw & Open Agent Ecosystem Index

A curated index of **OpenClaw / PicoClaw / NanoClaw forks, ports, runtimes, orchestration layers, and open-source agents**.

**Purpose:**
Create a single entry point that maps the ecosystem clearly, reduces fragmentation, and enables structured collaboration.

---

## 🎯 Vision

* **Unify instead of fragment**
* **Make implementations comparable**
* **Encourage structured contribution**
* **Highlight runtime diversity (Go, Rust, TS, Nim, C#, etc.)**
* **Track orchestration and deployment patterns**

# 📦 Inclusion Criteria

A repository qualifies if it:

* Is a fork, derivative, or conceptual extension of OpenClaw/PicoClaw/NanoClaw
* Provides runtime, deployment, orchestration, or infrastructure support
* Implements connectors, skills, or integration layers
* Clearly follows the “Claw” architectural philosophy

---

# 🤝 Contributing

When submitting a PR, include:

* Repository link
* Short one-line description
* Primary language / stack
* Category (Runtime / Port / Orchestration / Chat Agent / Deployment / Skills)
* Optional: maturity level, release frequency, distinctive features
